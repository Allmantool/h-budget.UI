# Angular PR Review Prompt

```text
Objective: Review the complete change against [base branch/commit].
Scope: Review only; do not edit unless explicitly asked.
Non-goals: speculative rewrites or compliance claims without evidence.
Required discovery: Read AGENTS.md and the Angular/Nx skill; inspect base/diff, affected code/tests/contracts, project configuration, and current worktree state.
Expected validation: Inspect relevant test/build evidence; identify missing validation rather than inventing it.
Self-review: Recheck every finding for severity, concrete impact, location, and task relevance.
Final response: Findings ordered by severity with precise evidence; questions/assumptions; positive coverage only where verified; validation gaps; summary.
```

Authority: [root instructions](../../AGENTS.md) and [Angular/Nx skill](../skills/angular-spa/SKILL.md).
